## Reporting a Vulnerability

Even though, there isn't a lot to make a vulnerability of there is still a chance of it happening. If you believe you have found one, report it immediately. Do not go, and share it to be abused. 
So where do I report it? We have 2 main ways:

| Where           | Contact Detail                                                                | Response Times     |
|:----------------|:------------------------------------------------------------------------------|:-------------------|
| Discord Server  | https://discord.gg/YmQs87ZTBB                                                 | Fast               |
| Email           | chamogelastos@chamosmp.net                                                    | Fast to Medium     |
| Github Advisory | [Here](https://github.com/SQD-Studios/ChamoItemSkins/security/advisories/new) | Fast (Reccomended) |
